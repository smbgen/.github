# smbgen

Official GitHub organization profile and community health defaults for `smbgen` repositories.

## What This Repository Is

This repository provides organization-wide defaults for:

- community standards
- issue and pull request workflows
- security reporting guidance
- contributor onboarding

GitHub automatically applies many of these files as defaults to public repositories in this organization when repository-specific files are not present.
