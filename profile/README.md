# smbgen

A CRM, CMS, and client portal with contact, booking, and payment interfaces.

Built on **Laravel 12** · **Livewire 3** · **Tailwind CSS** · **Alpine.js**

---

## What It Does

| Area | Capabilities |
|------|--------------|
| **CRM** | Client management, lead tracking, CSV import, activity logs, role-based access |
| **CMS** | Visual page builder, custom branding, blog with WYSIWYG editor, SEO tools, RSS |
| **Client Portal** | Magic link SSO, document sharing, messaging, package review flows |
| **Contact** | Lead capture forms, form-to-lead conversion, email delivery |
| **Booking** | Appointment scheduling, availability management, Google Calendar & Meet integration |
| **Payments** | Stripe invoicing, payment collection, webhook handling, refunds |
| **AI** | Claude-powered blog generation, SEO metadata, content analysis, usage tracking |

---

## Tech Stack

- **Backend** — Laravel 12, PHP 8.4
- **Frontend** — Livewire 3, Alpine.js, Tailwind CSS 3
- **Storage** — S3 / R2 with local fallback
- **Payments** — Stripe via Laravel Cashier
- **Testing** — Pest PHP
- **Integrations** — Google OAuth, Google Calendar, Google Drive, Anthropic Claude, Stripe
